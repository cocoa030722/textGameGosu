
텍스트게임 커맨드 사양
1.정보:현재 층의 정보를 볼 수 있음
2.탐험:층을 탐색하여 랜덤한 결과를 얻음
-일반 층의 경우 적, 아이템, 통로 발견 가능
--각 요소는 (개별 요소(적, 아이템, 통로)의 개수)/(전체 요소(적+아이템+통로)의 개수)의 확률로 나타남
--나타난 요소는 리스트에서 제거함
---적 나타남:공격, 도주, 아이템 커맨드
---아이템 나타남:자동으로 습득
---통로 나타남:다음 층 이동, 현재 층 계속 탐색 커맨드
-보스 층의 경우 확정 전투->통로 발견
3.싸움:적과의 전투
-일반전과 보스전은 동일한 전투 시스템을 사용, 보스전은 전투 전후로 이벤트를 띄움
-데미지=공격력-방어력
-둘 중 하나의 hp가 0 초과일 동안
--플레이어 공격->적 hp가 0 이하면 종료
--적 공격->플레이어 hp가 0 이하면 종료
-승리 시
--승리한 적을 맵 정보에서 배제
-패배 시:게임 종료

4.종료:종료

저항도, 순응도 관리
-저항도:적극적 저항, 소극적 협력에 관여
--너무 낮으면 멘탈 이상/너무 높으면 항명
--관리:처벌, 억제 마법(단기적 감소, 장기적 폭발 가능성)
-순응도:적극적 협력에 관여
--너무 낮으면 아무 행동도 안 함/높을수록 좋음
--관리:보상
-기타 커맨드
--가만히 방치 시 자동으로 순응도 감소, 저항도 증가
--피해나 상태이상 입은 채 방치 시 순응도 감소, 저항도 증가 가속
--장비 업그레이드:전투력 증가, 순응도/저항도 동시 증가
--영혼석:동료 편입 때 자동으로 부착, 플레이어 마력을 투입하여 저항도 감소, 순응도 증가
--감시 장치:항명 이벤트 시 디버프를 걸고 시작
--지배 마법:저항도 0, 순응도 100으로 고정
